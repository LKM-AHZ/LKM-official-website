## 欢迎加入理科迷

理科迷是一个由普罗大众组成的科技爱好者社区，创立于 2014 年。

### 第一步：加入群组

你可以通过理科迷官方 QQ 群组加入我们。目前设有初中群和高中群两个分群。

### 第二步：了解社区

阅读社区规范，了解各个部门的工作内容。

### 第三步：参与活动

关注官方宣传账号和群公告，及时获取活动信息。
