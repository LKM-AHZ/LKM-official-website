### 欢迎加入理科迷

经过官网上各种各样的介绍，想必您已经对于该组织已经有了大概的了解，我们拥有庞大，且范围细致的专业分类和年级分段群QQ社群，并且现在有了b站的官方账号，传播更多知识。但是这对于我们来说还是只是一个微不足道的一小步，我们需要像您一样的人加入我们！所以这一篇是介绍如何进行理科迷社区建设，如何互建共享，如何加入七月建设团队！

### 第一步：七月团队？

我们最早的社群是2014年8月1号进行成立，筹备呢是7月份进行的，所以叫做七月团队，当然了，这个小巧思是我们的创始人所起的，虽然组织在疫情期间之后有些没落，但是在又开始重组工作进行了沿用。在过去的组织架构里，组织分为总务组、群务组和顾问组，这三个不同的组拥有不同的职权。但对于现在来说，组织现在在激剧的扩张，所以又对于组织架构进行了调整和扩大。目前我们分为主要负责管理和规划的管理团队，以及负责落实计划的项目团队，这个我们会依次进行介绍，您可以按照自己的能力和专长以及意愿，加入我们！

## 七月管理团队架构
以下这些是我们目前组织的架构以及代表的颜色：
❤️总务部
💙群务部
💚新闻办
🧡活动部
💛顾问团
💜专业委员会
🖤技术委员会
管理团队成员，会在社群里显示有这样的牌子。
# 总务部
总务部原为总务组，总务部是作为七月团队最核心的决策和管理的部门。决策着我们会有什么样的规划和方向，管理各其他下属部门，是属于管理与被管理的关系。其他部门的负责人均为总务部一员。有时候需要统筹规划的东西很多，涉及到很多的部门协调的问题，所以一个总务组成员会管理很多事务，涉猎很多部门。但由于现在组织确实在急速的扩张，有时候管理团队跟不上，现在我们也非常急需，有更多能够统筹规划，对于某一部门进行负责任的”理科迷”。这样子就能减轻总务部门负担，而放在更重要的规划和执行，并非管理这种事情。
# 群务部
群务部原为群务组，负责管理负责各平台的管理与调动氛围，把控风向，是属于总务组的后备成员，受总务组协调，是属于下细分支。目前我们有社群的平台只有腾讯群组，但等到后面宣传到位，平台还可以扩展为贴吧等等不同的平台。到时候我们还会设置成专务组，进行专门的管理和协调。

我们目前对于群务部门的要求是，能够向总务部看齐。就先从QQ社群来说，我们希望群务组能够管好一个群，协助总务部、新闻办、活动部，能够顺利在社群里面，开展各种项目的落实和活动规划。毕竟只是把群规划好，不仅活跃，又有学习的风气，有很多人能够参加活动，参与组织建设，也是件很困难的事情哦～所以在某种意义上说，也需要有统筹和规划的能力，但是对于总务部那种大的组织管理来说，群务部是很好的可以进行适应和学习的部门，而且对于其他部门来说入门的门槛也是比较低的，它只需要你有一个，愿意进行社区建设的心，所以非常亲民。
# 新闻办
新闻社，会负责理科迷的对内外科技交流、公关与宣发工作，目前分有宣传组、制作组，以及科普组，如果有更多的人加入我们，我们就可以实现更多对于新闻办所规划的话需要发挥的职能，组建更多职能小组和旗下项目团队。

宣传组最主要的是对于宣传平台的运营，比如说办活动、发布科技类的相关新闻、文章和视频，确保宣传账号粉丝活跃，能够扩大更多的影响力。在这个组工作需要习惯性的在某一平台进行上线，不怕生，会关注一些比较火，产出非常不错的UP进行谈合作相关事宜;经常关注与科技相关的新闻，要及时的进行传播;要会文案编辑;会有更多想法运营好官方账号等等内容。适合有某一专长发展，经常阅读前沿科技文献的人，又或者说是对于平台运营熟悉，有所了解的人进行加入。

制作组顾名思义，就是制作各种视频剪辑和特效、做宣传海报、视频拍摄等等，做宣传与实际产出的制作，做保障工作。在这个组工作，就需要有熟练使用应用，如PS、AE、3D、可视化工具、等作图和视频剪辑，需要会进行UI界面设计，特效，剪辑等能力，有其一专长即可加入。

科普组最主要的是对于科普内容的产出，如文章、视频等等内容。也可以称得上是对于宣传视频内容上，负责策划的小组，这也是我们宣传，最会优先考虑进行传播的。需要自己对于某一方面熟悉，并知道如何进行撰写，和制作会让大家喜欢上，并能够领略到科技的美。

外语组，其实我们也有想过要获取更多的宣传影响力，所以外语组对我们来说是非常有所必要的，欢迎拥有两种以上多语言人才加入我们！负责我们各种科普内容，实际内容的产出进行翻译，同时也会负责组建翻译团队产出并促进国内外科技相关的交流，以及国际平台的开拓。目前该组的成员，主要是在国际科技语言交流学习社进行招募，是属于在未来会规划当中确定好的小组。
# 活动部
下分为常务组、策划组、执行组，内容组，最核心组为常务组，并管其他组员的筹备、策划与执行工作，下管理专业委员会。未来以后会组织新闻社与专业委员会一起成立相应工作组，翻译最新前沿科普内容，如负责最新国外的讲课讲座、论文、著作等等，助力知识传播。能够从执行到策划。

策划组就是负责与活动的策划，执行就是负责与活劳动的执行，内容负责充实活动里面的内容。保证一次活动能够顺利进行的为活动负责人。鼓励策划组和执行组像活动负责人看齐，会把创意到落地一条龙服务，拥有同等的创意以及执行的能力。

活动部要紧跟着平台走，如QQ社群以及b站等宣传账号，为单独和特定的群体能够举办活跃气氛的游戏或者活动，保证群体的活跃。

唯一的要求就是会写活动策划案以及，有高效、准时的执行能力，以及应急的能力。我们鼓励大家自己有什么想要在社区里开办的活动，自己办，一起众乐乐。该部门只要你有创意、有想法，有任何想要在社区尝试开办的活动，就可以加入我们！

# 财政处
财政处是未来会建立的部门，虽然说目前总务组的威望和信服力是很高的，但是顶着需要对大家所募捐的钱负责，公示财政每一笔收入以及支出，财政处会给出，支出每一笔相应的合理性的报告。需要对于组织经济这一方面进行考量建议、帮助和监督。同时也会加强对于组织的政治建设，让更多人接触并学习与马列毛知识，不鼓励大家当网左，当愤青，鼓励学习与实践。适合有信仰、有目标、有方向、思想正确，有大量知识理论储备，的同志们加入做部门建设。适合脚踏实地的理想主义者进行加入。

# 顾问团
顾问团原为顾问组，将有本体和非本体，主要包含的职能为技术支持与咨询、法律顾问、对外交流学习组织等，如果未来有的话，就会分为技术顾问组，法律顾问组等部分。这个部门在组织里来说是最为清闲的一个部门，毕竟只要提提意见，接受一下组织的咨询就可以了～适合有技术能力、有想法、有目光、有眼界，但是碍于时间不够，不能顾及组织上的事物的人加入。
# 专业委员会
专业委员会的成分，主要是各平台当中，有相关专业水平、热爱科普事业为主、参加活动为其次的人员。职能是负责产出科普视频、公开课、文章等等形式，将知识传播于群众。我们现有的分组就有了数学组，物理组，化学组和生物组等等各部分的专业组成员，他们负责很多活动的填充、以及科普内容上面的指导，并且解答社群里面人对于专业上面的困惑。

随着社群的扩大，我们也会更多的招揽相关的人才，在更多方向上面，都可以对于各种各样方面上面感兴趣的人，提供上自己的指导建议以及学习上的帮助。专业委员会鼓励学生、教师、科研工作者，科普工作者、有更多来自各行各业的技术人员、各行各业的从业者加入。拥有想要在某一方面进行深钻或各种专业深度知识储备的人越多越好。

我们专业委员会有专业水平，半专业组和非专业组水平有所区分。

专业组是指在某一方面水平已经达到很高的阶段，有指导他人纠错的能力。

半专业水平是对于普通学生来说，拥有的知识储备已经高上了一个程度，但是他对于深奥的东西，人所有困惑，或者说是没有全面的掌握。

非专业组水平就是说是目前已经在学习的路上，比入门好一点，但是仍然是一个学习在不稳固的情况和基础上，知识和经验都较少。

专业委员会同样是，会按照大家现有掌握的水平进行区分的～所以大家不必担心加入会显得自己没有能力，相反，比起自卑来说，更多的跟专业性强的人学习，是一个使自己快速成长更好的一个选择。

# 技术委员会
技术委员会目前负责理科迷技术支持，网站的搭建与维护，如果以后有可能，会向对外合作开展学习组织交流论坛，或是理科迷属于自己的软件。除此以外的话，如果还需要其他技术上面支持的需要，到时候也会全部拉入技术委员会，进行详细的分组。技术委员会的人现在也很少，如果你有以下的技能其一：搭建、运维、建立防火墙、AI、小机器人、小程序、论坛、网站，提供活动支持等技术就即可加入。

目前我们在建设实际功能性网站，需要会前后端交互的人才，现在加入就可以接触到项目哦～

## 七月项目团队架构
接下来我们要着重介绍一下，项目团队架构，这是一个全新的独立的一个架构，它是隶属于管理团队下面的一个庞大的分细。主要的发起在于理科迷的成员，想要共建什么样的项目而成立。这个是只要有能力，有野心，有想法，有执行能力，把自己的想法变为现实，就可以成立相应的项目小组。

总务部门呢，也有很多的想法比如:说科普项目、期刊项目、学术讲座组建工作、自媒体…………但是我们也深深的明白，只靠自己有这个想法，但是没有时间落实是一个难办的过程，所以也希望广大有志之士，能够进行执行和落实，一起能够帮助更多的人共建共享！

我们就先介绍现有的两个项目：一个就是学习资料编写项目组，还有就是科普产出项目组。

# 学习资料编写项目组
这个项目是由群里面的大雄提出，为主要负责人，目前科目为初高中数理化生，未来会补充英语，历史的内容。我们目前主攻的科目是数学，大雄先做的是初高阶段的数学，最主要的它的特色是一本通读资料，对于学习能力弱的人，以及落后的人，他是讲的非常的明白的，每一个小的知识点都会讲到位，由浅到深，可以和初高中讲课教材同时服用。
LittlePony得负责的是高数部分的负责人，他发现现今的教材很多都是以刷题为主，很少能够真正培养学生对数学背后原理的思考。希望能够建立一个教材，能让数学爱好者除了能够精通知识之外，还能够了解数学背后的原理，希望能够数学变得更有趣，能够让更多人喜欢上数学，而愿意进行深造。

LittlePony对微积分这一类是相当的熟悉，会优先搞高数微积分这一块。矩阵和向量这部分，要找对此熟悉的数学竞赛生，因为的确是他们对于这方面会更加熟悉。如果愿意来帮忙的话，可以来加入学习资料编写组。

虽然都是数学教材，是更多的为了高考做准备，不会讲高考体制以外的其他事务，适合需要进行备战的人;而LittlePony则是为了数学爱好者们精通知识，了解数学背后的原理，逻辑，并不是为了应试做准备，所以适合愿意在这一块进行深造的，并想挑起数学兴趣的人进行学习和阅读。

我们学习资料编写项目组还是很缺人的，因为我们希望有更多的有更多知识储备的人、学生，教师加入我们，一起进行项目的落实。

# 科普产出项目组
我们科普产出项目组是由小星染，但由于科普产出，这一个也是一直总务组想做的事情，但是还是由于缺乏人手而没有成立，这种情况是由他到来之后好的多。目前我们想要做的是自媒体上面科普内容的产出，目前我们想要做的是自媒体上面科普内容的产出，希望能够以此更多的人会对此感兴趣，然后就能加入到我们。如果您在这个地方有一定的建树，欢迎您给我们项目组给出相应的建议！

## 各部门之间的关系
我们所有的部门现在都是属于平级管理，加入一个部门的群，再加入一个理事会的群。权利会下放到部门负责人进行分配，各部门的事情由部门负责人进行统一调度，但如果是属于七月团队需要共同考量的事情，就会在理事会群里面进行说。虽然说大家都是为爱发电，但依旧要对于组织的事情负责任，也不能拥有在网上当赛博皇帝的意识。

## 组织内部的规章制度
目前我们组织内部的规章制度有两份，一份的就是管理规范守则，第二个就是七月团队理事会公约。一个是对于组织需要做的事情有确切的要求，二是对于团队内部风气，纪律有一定的要求。这两份都要进行到合理的落实。这两部分规章制度，都会进行上传。后续还会再印制各部门需要的工作守则，会教大家具体如何能够做好工作组的事务。
